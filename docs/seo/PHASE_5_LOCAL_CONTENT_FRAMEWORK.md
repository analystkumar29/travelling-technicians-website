## CONTENT CREATION GUIDELINES

### Format Requirements
- **Word Count**: 400-600 words per section
- **Keywords**: Include 2-3 primary keywords naturally
- **Links**: Internal links to service pages (2-3 per section)
- **Structure**: H3 headings, 2-3 paragraphs, bullet points where appropriate
- **Tone**: Professional, friendly, locally-aware

### SEO Best Practices
- Use neighborhood names naturally in text
- Include seasonal references (for seasonal sections)
- Add specific local landmarks or institutions
- Use long-tail keywords (e.g., "iPhone screen repair Yaletown")
- Link to service pages contextually

### Content Calendar
- **Week 1**: Vancouver content (all 5 sections)
- **Week 2**: Burnaby + Coquitlam content
- **Week 3**: Richmond + North Vancouver content
- **Week 4**: Review, optimize, implement
